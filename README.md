# MiniCoder (English)

**MiniCoder** is an interactive educational web application designed to teach children and beginners the fundamentals of programming in a fun and accessible way. Using a simplified language inspired by BASIC (called MiniQB) and a series of guided lessons, users learn key concepts such as variables, loops, conditionals, and algorithmic logic through hands-on practice.

*   **üéì Guided Interactive Lessons:** A 13-lesson course covering everything from the most basic concepts to building a complete game.

*   **ü§ñ 2D Logic Simulators:** Interactive missions with a robot (MiniBot) to teach algorithmic and sequential thinking in a visual and entertaining way.
*   **üìñ Manual & Quick Help:** Comprehensive documentation for the MiniQB language and a contextual help panel that activates as you type in the editor.
*   **üì± PWA Support for Mobile:** The application is a Progressive Web App, meaning it can be "installed" on your phone's home screen and works offline.

## üìö Lessons Overview

1.  **The Basics:** What is programming? Introduction to logical thinking.
2.  **Logic I:** Learn to give precise instructions with the robot simulator.
3.  **Logic II:** Understand boolean thinking (true/false).
4.  **Memory & Conversation:** Discover variables and how to interact with the user.
5.  **The Data Workshop:** Master mathematical and text operations.
6.  **Decisions (IF):** Teach your program to make decisions.
7.  **Loops I (FOR):** Learn to repeat tasks an exact number of times.
8.  **Loops II (DO):** Master repetitions until a condition is met.
9.  **Project - Text Adventure:** Create your first video game.
10. **Final Mission - Star Odyssey:** Command a spaceship in a complex game.
11. **Extra - Music & Sound:** Add sound effects and melodies to your creations.
12. **Extra - Arrays:** Learn to handle lists of data.
13. **Project - Tic-Tac-Toe:** Build the classic game using arrays.
